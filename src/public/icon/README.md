# Icon Placeholder

This directory should contain extension icons in the following sizes:

- 16x16 (toolbar)
- 48x48 (extension management)
- 128x128 (Chrome Web Store)

You can generate these using an online tool or image editor.

For now, WXT will use default icons. To add custom icons:

1. Create PNG files: 16.png, 48.png, 128.png
2. Place them in this directory
3. WXT will automatically include them in the manifest
